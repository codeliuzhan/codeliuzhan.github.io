2021/04/27 Oracle第八天

前言

今天笔记对数值型函数和日期型函数做一个总结。

数值型函数

1.取绝对值

ABS(NUM):获取数值NUM对应的绝对值。
该函数通常不直接对某数值进行取绝对值，而是对于一系列的计算结果获取绝对值。

SELECT ABS(3.45),ABS(-4.56) FROM DUAL;
SELECT SAL,COMM,SAL-COMM,ABS(SAL-COMM) FROM EMP;
SELECT ABS(5),ABS(-4) FROM DUAL;
SELECT ABS('A'),ABS(-4) FROM DUAL;

2.向上取整与向下取整

  • CEIL(NUM) ：向上取整，获取离NUM最近的两个整数中较大的那个数
  • FLOOR(NUM)：向下取整，获取离NUM最近的两个整数中较小的那个数
    SELECT CEIL(2.56),FLOOR(2.56) FROM DUAL;
SELECT CEIL(5),FLOOR(5) FROM DUAL;
    注意：
  1. 若NUM本身即为整数，则无论向上取整还是向下取整，返回结果都为NUM本身
  2. 无论NUM是正数还是负数，所谓上，即数轴指向的方向，所谓下，即数轴背向的方向
    SELECT CEIL(-2.56),FLOOR(-2.56) FROM DUAL;

SELECT CEIL(1.23,2),FLOOR(1.23,2) FROM DUAL;–参数个数无效
sELECT CEIL(‘1.2a5’),FLOOR(‘1.2a5’) FROM DUAL;–无效数字
SELECT CEIL(‘A’),FLOOR(‘A’) FROM DUAL;–无效数字


#### 3.取余
MOD(X,Y)：X是被除数，Y是除数，计算Y整除X，获取计算结果中的余数。（不是商！不是商！不是商！）
MOD函数对于纯数学范畴的使用很少，但智慧的程序员们却因此提出了“分库分表”的概念：
将数据按照一定规则均分到不同的数据库（表）中，变串行为并行，以此来提高代码和程序的运行效率。
```shell
SELECT MOD(13,5) FROM DUAL;
select * from STUDENT_ALL;

SELECT STD_NO,CASE WHEN MOD(STD_NO,4)=0 THEN 'A'
                   WHEN MOD(STD_NO,4)=1 THEN 'B'
                   WHEN MOD(STD_NO,4)=2 THEN 'C'
                   ELSE 'D'
               END 
  FROM STUDENT_ALL;

4.四舍五入

ROUND(NUM[,P])：对数值NUM进行四舍五入，其中参数P决定了四舍五入的精度
基础用法

SELECT ROUND(5.6789,2),ROUND(5.4321,2) FROM DUAL;
SELECT ROUND(12345.6789,1) FROM DUAL;

精度为0或者精度省略

SELECT ROUND(5.6789,0),ROUND(5.6789) FROM DUAL;

负数的四舍五入

SELECT ROUND(-5.4),ROUND(-5.6) FROM DUAL;

精度为负数

SELECT ROUND(45.321,-1),ROUND(45.321,-2),ROUND(55.321,-2),ROUND(0055.321,-3) FROM DUAL;

精度不为整数

SELECT ROUND(5.4321,2.22222),ROUND(5.4321,2.99999) FROM DUAL;
--按照整数位进行计算

5.截断

TRUNC(NUM[,P]) ：对数值NUM进行截断，其中参数P决定了截断的精度

基础用法

SELECT TRUNC(4.54321,2),TRUNC(4.56789,2) FROM DUAL;

精度为0或精度省略

SELECT TRUNC(4.54321,0),TRUNC(4.54321) FROM DUAL;

负数的截取

SELECT TRUNC(-1.4),TRUNC(-1.6) FROM DUAL;

精度为负数

SELECT TRUNC(45.321,-1),TRUNC(045.321,-2),TRUNC(055.321,-2),TRUNC(0055.321,-3) FROM DUAL;

精度不为整数

SELECT TRUNC(5.4321,2.22222),TRUNC(5.4321,2.99999) FROM DUAL;

1.关于日期的四舍五入与截断

ROUND() 函数与 TRUNC()函数
除了可以作用在数值上，也可以作用在日期上，对于日期也同样有四舍五入或截断的需求

对于四舍五入，
在数值方面依靠数值5作为分界线，达不到5即被舍去，达到5即向前进1。

在日期方面，四舍五入同样需要一个分界线，该分界线在不同的时间单位中并不相同，原则上是取居中的位置作为分界线。

2.获取月份差值

MONTHS_BETWEEN(d1,d2) :求D1和D2之间相差几个月（d1>d2）

常规用法：给定两个日期求两日期之间的月份差

  1. 计算从元旦到今天经过了几个月（非整数月）

    SELECT TRUNC(MONTHS_BETWEEN(TRUNC(SYSDATE,'DD'),TO_DATE('20210101','YYYYMMDD'))) FROM DUAL;
SELECT 26/31 FROM DUAL;

  2. 计算从元旦到今天经过了几个月（整数月）

    SELECT TRUNC(MONTHS_BETWEEN(TRUNC(SYSDATE,'DD'),TO_DATE('20210101','YYYYMMDD'))) FROM DUAL;

SELECT ABS(TRUNC(MONTHS_BETWEEN(TO_DATE(‘20210101’,’YYYYMMDD’),TRUNC(SYSDATE,’DD’)))) FROM DUAL;

SELECT TRUNC(SYSDATE,’MM’),MONTHS_BETWEEN(TRUNC(SYSDATE,’MM’),TO_DATE(‘20210101’,’YYYYMMDD’)) FROM DUAL;

SELECT MONTHS_BETWEEN(TO_DATE(‘20210430’,’YYYYMMDD’),TO_DATE(‘20210228’,’YYYYMMDD’)) FROM DUAL;

3.获取月末日期

LAST_DAY(D)：获取指定日期对应的当月最后一天，即月末日期

SELECT LAST_DAY(SYSDATE) FROM DUAL;

举例：
获取当月最后一天。

SELECT LAST_DAY(SYSDATE) FROM DUAL;

查询某月份的最大天数

SELECT LAST_DAY(SYSDATE+30) FROM DUAL;

SELECT TO_CHAR(LAST_DAY(SYSDATE+30),'DD') FROM DUAL;

获取给定两日期的月份差（整数月）

SELECT MONTHS_BETWEEN(LAST_DAY(TO_DATE('20210415','YYYYMMDD')),LAST_DAY(TO_DATE('20210202','YYYYMMDD'))) FROM DUAL;

查询某日期对应的年份是平年还是闰年

--通过判断2月的最后一天判断平年闰年
SELECT  DDDD,R,CASE WHEN TO_CHAR(LAST_DAY(TO_DATE(SUBSTR(DDDD,1,4)||'0205','YYYYMMDD')),'DD')='29'
             THEN '闰年'
             else '平年'
         END
   FROM DAY_TEST;

SELECT * FROM DAY_TEST FOR UPDATE;

--通过判断全年的最后一天判断平年或闰年
SELECT DDDD,R,CASE WHEN TO_CHAR(TO_DATE(SUBSTR(DDDD,1,4)||'1231','YYYYMMDD'),'DDD')=366
                   THEN '闰年'

--通过年份判断平年或闰年

4.月份加减

ADD_MONTHS(D,N) :在D日期的基础上加N个月。N可为正可为负。
基础用法：

SELECT SYSDATE,ADD_MONTHS(SYSDATE,2) FROM DUAL;

N为负数即为在D日期的基础上减去几个月

SELECT SYSDATE+2,ADD_MONTHS(SYSDATE+2,-2) FROM DUAL;

特殊情况：

1）如果N是小数–以整数部分计算

SELECT SYSDATE,ADD_MONTHS(SYSDATE,2.5) FROM DUAL;

2）如果DD是月末

SELECT SYSDATE+2,ADD_MONTHS(SYSDATE+2,1) FROM DUAL;

3）如果DD不在计算后的月份中

SELECT SYSDATE+2,ADD_MONTHS(SYSDATE+2,-2) FROM DUAL;

例题：查询比SMITH早入职两个月的员工信息

SELECT * FROM EMP WHERE HIREDATE=ADD_MONTHS((SELECT HIREDATE FROM EMP WHERE ENAME='KING'),2)

思考：
返回日期对应的当月第一天

--1.截断
SELECT DDD,TRUNC(DDD,'MM') FROM DAY_TEST;  --主要针对日期

--2.截取+连接
SELECT DDDD,SUBSTR(DDDD,1,7)||'/01' FROM DAY_TEST;  --主要针对字符

5.获取下个周几

NEXT_DAY (D,W)：给定日期D，和星期W，获取D日期之后的下一个星期W

SELECT TO_DATE('20210428','YYYYMMDD'),
       NEXT_DAY(TO_DATE('20210428','YYYYMMDD'),'星期五') 
  FROM DUAL;
思考题：

给定任意日期，如何获取下个周的星期五

SELECT NEXT_DAY(TO_DATE('20210428', 'YYYYMMDD'),'星期五')+7 from DUAL;
SELECT NEXT_DAY(TRUNC(TO_DATE('20210426', 'YYYYMMDD'),'IW')+7,'星期五') from DUAL;

给定任意日期，如何获取下个周的星期几

SELECT NEXT_DAY(TRUNC(TO_DATE('20210426', 'YYYYMMDD'),'IW')+6,'星期一') from DUAL;

其他函数

1.条件取值

DECODE(EXPR,VALUE1,RESULT1,VALUE2,RESULT2,…,DEF_RESULT)

其实效果即等同于

CASE  EXPR
     WHEN VALUE1  THEN RESULT1
     WHEN VALUE2  THEN RESULT2
          …
    ELSE DEF_RESULT
 END

举例：
使用decode函数，职位是分析员的，工资+1000；职位是经理的，工资+800；
职位是其它的，工资+400

SELECT DECODE(JOB,'ANALYST',SAL+1000,'MANAGER',SAL+800,SAL+400) FROM EMP;

SELECT * FROM EMP;
SELECT CASE JOB
            WHEN 'ANALYST' THEN SAL+1000
            WHEN 'MANAGER' THEN SAL+800
            ELSE SAL+400
        END
   FROM EMP;

2.空值的赋值

NVL(EXPR,VALUE):如果EXPR是空，返回一个VALUE，如果EXPR不为空，返回EXPR本身

SELECT COMM,NVL(COMM,0) FROM EMP;
SELECT COMM+SAL,NVL(COMM+SAL,0) FROM EMP;
SELECT NVL(NULL,'AAA') FROM DUAL;
SELECT NVL(NULL,SYSDATE) FROM DUAL;

NVL2(EXPR1,EXPR2,EXPR3):expr1不为NULL，返回expr2；expr1为NULL，返回expr3

SELECT NVL2(SAL+COMM,SAL+COMM,SAL)FROM EMP;

3.去重

DISTINCT COL_LIST   :
SELECT DISTINCT COL_LIST FROM TB_NAME …

对COL_LIST范围内的字段进行去重

举例（以下查询结果全部要求去重）：

获取公司的所有岗位

SELECT DISTINCT JOB FROM EMP;

获取公司的所有部门

SELECT DISTINCT DEPTNO FROM EMP;

获取各部门的各岗位

SELECT DISTINCT DEPTNO,JOB FROM EMP;

SELECT DEPTNO,JOB FROM EMP GROUP BY DEPTNO,JOB;

SELECT DEPTNO FROM EMP GROUP BY DEPTNO;
